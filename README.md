Smart Workspace Manager

Smart Workspace Manager, bilgisayarınızdaki uygulamaları ve sistem kaynaklarını yönetmenizi sağlayan gelişmiş bir araçtır. Farklı kullanım senaryolarına uygun modlar oluşturabilir, RAM ve CPU tüketen uygulamaları izleyebilir ve istenmeyen uygulamaları kapatabilirsiniz.

Özellikler

Modlar ile Çalışma:
Farklı senaryolar için modlar oluşturun (örn. Oyun Modu, Geliştirici Modu). Modlar aktif edildiğinde belirlenen uygulamalar otomatik olarak açılır veya kapanır.

Gerçek Zamanlı Sistem İzleme:
CPU, RAM ve GPU kullanımını animasyonlu grafiklerle takip edin.

Seçilebilir Uygulamalar:
Çalışan uygulamalar listesinden kapatılacak uygulamaları seçin ve yeni mod oluştururken açılacak uygulamaları ekleyin.

Kolay Mod Yönetimi:
Modları oluşturun, düzenleyin veya silin. Aktif modları tek tıkla başlatıp durdurabilirsiniz.

Güvenlik Filtreleri:
Windows'un kritik süreçleri otomatik olarak filtrelenir, yanlışlıkla kapatılması önlenir.

Kullanıcı Dostu Arayüz:
Modern, temiz ve profesyonel tasarım ile hızlı ve kolay kullanım.

Kurulum

Depoyu klonlayın:
git clone https://github.com/kemalsarac/smart_workspace_manager.git

Flutter ortamını hazırlayın:
Flutter Kurulum Rehberi

Proje dizinine gidin ve bağımlılıkları yükleyin:
cd smart_workspace_manager
flutter pub get

Uygulamayı çalıştırın:
flutter run

Not: Windows ortamında çalıştırırken Developer Mode aktif olmalıdır.

Kullanım

Mod Oluşturma:
"Mod Ekle" butonuna tıklayın, açılan sayfada kapatılacak ve açılacak uygulamaları seçin, mod adını yazın ve kaydedin.

Modları Çalıştırma:
Ana sayfadaki mod kutucuklarından istediğinizi seçin ve çalıştır/durdur ikonunu kullanın.

Mod Düzenleme veya Silme:
Mod kutucuklarının üstündeki düzenle ve sil ikonları ile modları yönetebilirsiniz.

English Version
Smart Workspace Manager

Smart Workspace Manager is an advanced tool that helps you manage applications and system resources on your computer. You can create custom modes for different scenarios, monitor RAM and CPU usage, and close unwanted applications automatically.

Features

Mode Management:
Create modes for different scenarios (e.g., Game Mode, Dev Mode). Selected applications will automatically open or close when the mode is activated.

Real-Time System Monitoring:
Track CPU, RAM, and GPU usage with animated graphs.

Selectable Applications:
Choose which running applications to close and add which applications should open in a new mode.

Easy Mode Control:
Create, edit, or delete modes. Start or stop active modes with a single click.

Security Filters:
Critical Windows processes are automatically filtered to prevent accidental closure.

User-Friendly Interface:
Clean, modern, and professional design for quick and efficient use.

Installation

Clone the repository:
git clone https://github.com/kemalsarac/smart_workspace_manager.git

Prepare your Flutter environment:
Flutter Installation Guide

Navigate to the project folder and get dependencies:
cd smart_workspace_manager
flutter pub get

Run the application:
flutter run

Note: Enable Developer Mode when running on Windows.

Usage

Create a Mode:
Click "Add Mode", select applications to close and open, enter a mode name, and save.

Run Modes:
Select a mode from the main page and use the play/stop icon to activate or deactivate it.

Edit or Delete Modes:
Use the edit and delete icons on the mode cards to manage them.
